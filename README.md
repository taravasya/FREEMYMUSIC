# FREEMYMUSIC
Python скрипт, для простого переноса / импорта / экспорта, содержимого плейлистов, с некоторых стриминговых сервисов.  

Этот скрипт используется вместе с python-tidal https://github.com/tamland/python-tidal

Инструкция по переносу треков из Spotify в Tidal:  
---

Для наглядности восприятия части производимых действий, обращайтесь к демо-видео:  
https://www.youtube.com/watch?v=0EFJT80o-9w  
Файл конфигураций, ни в коем случае не редактировать в виндосовском блокноте. Для этой цели рекомендую Notepad++, Sublime Text...

----

Для программного доступа к плейлистам Spotify, необходимо создать WEB-приложение. Для этого нужно войти со своей учётной записью вот тут:  
https://developer.spotify.com/dashboard/applications  
Согласиться с условиями и создать WEB-приложение, нажав на кнопку:
>CREATE AN APP  

Задать приложению любое имя. Поставить две галочки и подтвердить его создание. После создания приложения, на открывшейся странице, вы увидите строку похожую на это:
>CLIENT ID 81a7f24bfd413ccfb4ffd22b48012cf

и ниже надпись:

>SHOW CLIEN SECRET

...кликнув по которой, Вы увидите ещё одну строчку с абракадаброй. Обе эти абракадабры, нам понадобятся позже.

Так же нам понадобится ID плейлиста в Spotify. Что бы получить его, нужно открыть этот плейлист в браузере, и скопировать из адресной строки ещё одну абракадабру )))). Например вот адрес моего плейлиста Микс Дня 3  
https://open.spotify.com/playlist/37i9dQZF1E39YIECalsJAz  
Здесь его ID:
>37i9dQZF1E39YIECalsJAz

то-есть всё, что после последнего слеша.

Установите Notepad++ или Sublime Text.

Скачайте исходный архив, в котором содержаться все необходимые для работы скрипта библиотеки, а так же сам мой скрипт в изначальной версии:  
https://mega.nz/file/3BhSDIKY#ou9_6shYheDdkZdqh8_nH3USHSPkeGRJObnwF-Vf3rc  
Распакуйте этот архив в любое удобное для Вас место, с учётом того, что в пути к распакованным файлам, не должно быть кириллицы.  
Затем скачайте последний релиз моего скрипта:  
https://github.com/taravasya/FREEMYMUSIC/releases/latest  
И распакуйте его содержимое в папку *freemymusic* с подтверждением замены файлов.

В распакованной папке, найдите файл MYSETTINGS.py (в проводнике Windows, возможно не будет отображаться расширение файла и тогда он будет выглядеть как MYSETTINGS ), нажмите на нём правой клавишей мыши, и выберите пункт, “Открыть с помощью”, и далее найдите Notepad++ (это действие наглядно видно на моём демонстрационном видео).

В файле MYSETTINGS.py в принципе всё лаконично. Нужно заполнить логин/пароль от тидал, client id и client secret от Spotify, а так же указать id плейлиста в Spotify и придумать название плейлиста который появится в Тидале и в который будут импортированы треки из спотифай. Если Вы не добавите название плейлиста, то он будет называться *FreeMyMusic*. Потом в интрефейсе Tidal, Вы сможете переименовать его как угодно.

Добавив все эти данные, сохраните файл.

Теперь осталось только запустить файл: !SPOTIFY_TO_TIDAL.bat (опять же, в проводнике, возможно не будет видно расширения и тогда он будет отображаться как !SPOTIFY_TO_TIDAL)

Если в открывшемся окне командной строки, вы увидите добавляемые названия треков, то всё в порядке и процесс пошёл. Когда окно закроется, можно открывать Tidal, и слушать новый плейлист.

Если окно командной строки сразу пропало, значит, просто попробуйте ещё раз. Если снова ничего не вышло, значит, что-то пошло не так. Тогда пишите мне, будем искать причины…
